# PingPong

Uma jogo de PingPong desenvolvido em Python utilizando a biblioteca pygame, permitindo interatividade e sistemas de gameficação
## Funcionalidades
- Sistema de pontuação
- Interação sonora
- Adversário funcional
- Sistema de pause
- Movimentação com teclado
- Interface gráfica simples

## Tecnologias utilizadas
- Python 3.11.9
- pygame 2.6.1

## Como executar o projeto
- Acesse o projeto pelo código fonte, ou [acesse por aqui](https://github.com/giulikarime/PingPong.git), e faça dowload do arquivo PingPong.ZIP
- Extraia os arquivos para sua máquina
- Abra a pasta "dist"
- Execute o Aplicativo: PingPong
